R(result)对象

| 参数名称    | 参数说明       | 备注      |
|---------|------------|---------|
| status  | 状态信息       | 请了解状态码  |
| message | 服务器返回的提示信息 | 可以为null |
| data    | 服务器返回的业务数据 | 业务对象    |



## 1.1 上传头像

- 请求路径 /image/upload

- 请求类型 POST

- 请求参数 (form-data)

| 参数名称 | 参数说明 | 备注   |
|------|------|------|
| file | 头像文件 | 不能为空 |

- 响应数据

| 参数名称     | 参数说明                      | 备注      |
|----------|---------------------------|---------|
| url      | 文件可访问的url地址(https://....) | 不能为null |
| filename | 文件名(没啥用)                  | 不能为null |



## 1.2 上传瓶子信息

- 请求路径 /bottle/upload

- 请求类型 POST

- 请求参数 (form-data)

| 参数名称     | 参数说明   | 备注      |
|----------|--------|---------|
| username | 用户名    | 不能为null |
| message  | 用户留言   | 不能为null |
| avatar   | 用户头像地址 | 可为null  |

- 响应数据 R对象

- 请求路径 /bottle/get

- 请求类型 GET

- 请求参数 无

- 响应数据 R对象

| 参数名称     | 参数说明   | 备注      |
|----------|--------|---------|
| username | 用户名    | 不能为null |
| message  | 用户留言   | 不能为null |
